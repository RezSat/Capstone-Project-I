# Dashboard UI

## Purpose

Admin dashboard interface for managing products, inventory, orders, and users.

## Layout

- Left sidebar navigation
- Top navigation bar
- Main content area

## Pages

| Page | Components |
|---|---|
| Overview | Stat cards, charts |
| Products | Data table, search, filters, pagination |
| Product Detail | Product info, variant list, image gallery |
| Product Edit | Form with tabs (details, variants, images, specs) |
| Categories | Data table, create/edit forms |
| Inventory | Stock table, adjustment form |
| Orders | Order table, status badges |
| Settings | Configuration form |

## Design Rules

- Clean, functional, data-focused
- White backgrounds with thin borders
- Orange for active states
- Role-based action visibility
