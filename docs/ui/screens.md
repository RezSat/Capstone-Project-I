# Storefront UI

## Purpose

Public-facing e-commerce interface for product browsing and purchasing.

## Pages

| Page | Key Components |
|---|---|
| Homepage | Hero section, category tiles, product rails |
| Product Detail | Image gallery, variant swatches, add to cart |
| Category | Product grid, filter sidebar |
| Checkout | Order summary, payment form |
| Account | Profile form, order history, addresses |

## Design Rules

- Image-first, product-focused
- Large product images on white backgrounds
- Orange accent for primary actions
- Oswald for headings, Inter for UI text
- Responsive across mobile and desktop
