# Product Listing UI

## Purpose

Product listing pages for both dashboard and storefront.

## Dashboard Product Listing

- Data table with columns: name, SKU, status, price, stock
- Search by name, SKU
- Activity filter (active/inactive/all)
- Sort by name, date, status
- Pagination
- Bulk actions (activate/deactivate)
- CSV export

## Storefront Product Listing

- Product grid with cards
- Category-based filtering
- Attribute-based filtering (size, color)
- Sort by price, popularity, newest
- Responsive grid (4 columns desktop, 2 mobile)
