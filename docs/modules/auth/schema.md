# Auth Schema

## Purpose

Validation schemas for authentication inputs.

## Schemas

| Schema | Purpose |
|---|---|
| `dashboard-forgot-password.schema.ts` | Email validation for password reset |
| `dashboard-reset-password.schema.ts` | New password validation |
| `customer-signup.schema.ts` | Customer registration validation |
| `customer-profile.schema.ts` | Profile update validation |
