# Products Schema

## Purpose

Validation schemas for product, category, and variant operations.

## Key Schemas

| Schema | Purpose |
|---|---|
| `admin-catalog.schema.ts` | Product listing filters and pagination |
| `admin-product-details.schema.ts` | Product create/update validation |
| `admin-variants.schema.ts` | Variant create/update validation |
| `product-image.schema.ts` | Image upload validation |
